Welcome to CoachAtlas. 

This is a sports coaching directory for athletes, organizations and coaches to better find and connect with each other. 

Site includes:
  Rails
  Postgres 
  Twitter/Bootstrap styling with HTML 5 and CSS 3
  Sass
  jQuery
  Coffeescript
  JavaScript
  CanCan
  Thin
  Paperclip
  Omniauth

  Tested with:
  Rspec
  Capybara
  Guard

The site is a work in progress MVP final project for Epicodus coding school. The landing page can be found at coachatlas.com
